# Web Knowledge Platform

A production-ready web intelligence platform that crawls, extracts, and queries knowledge from websites.

## Features
- Web crawling with politeness rules
- Knowledge extraction (entities, contacts, relations)
- MongoDB knowledge graph
- FastAPI backend
- React frontend
- Railway + Vercel deployment

## Quick Start
1. Push to GitHub
2. Deploy on Railway (add PostgreSQL, Redis, MongoDB)
3. Deploy frontend on Vercel
4. Set REACT_APP_API_URL environment variable

## API Endpoints
- GET /stats - System statistics
- POST /query - Search knowledge base
